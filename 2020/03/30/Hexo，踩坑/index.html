Hexo，踩坑

本地图片不显示

1.修改文件_config.yml，post_asset_folder: 设置为true。

2.输入hexo n "md文件名"，生成一个与md文件同名的文件夹。

3.安装hexo-asset-image，可以上传本地图片的插件，
输入npm install https://github.com/CodeFalling/hexo-asset-image --save。
(不使用npm install hexo-asset-image –save这个命令安装，版本号不对)

4.md用如下形式引用图片，![描述](md同名的文件夹/图片.jpg)。
