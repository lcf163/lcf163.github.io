Hexo，搭建

1.安装Node.js

官网链接：https://nodejs.org/，如下图所示：

2.安装Git

官网链接：https://git-scm.com/downloads，选择windows版本，下一步安装。
Git Bash相当于Linux中的终端窗口，用来打开终端。

3.安装Hexo

新建文件夹blog，打开Git Bash测试node、npm是否安装成功，如下图所示：

安装cnpm，提高下载速度

终端输入 npm install -g cnpm --registry=https://registry.npm.taobao.org，
测试cnpm是否安装成功。

安装Hexo

输入cnpm install -g hexo-cli，验证安装是否成功，输入hexo -v。
进入/blog文件夹，输入hexo init，如下图所示：

4.测试

常用命令如下：

输入hexo s，打开浏览器localhost:4000，
看到博客页面，说明本地环境搭建完成。
