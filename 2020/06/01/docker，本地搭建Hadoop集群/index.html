Docker，本地搭建Hadoop集群

系统环境

Ubuntu16.04

安装Docker

下载镜像

docker pull ubuntu:16.04

运行容器

docker run -ti ubuntu:16.04
不要关闭这个终端

安装JDK

tar -zxvf jdk-8u11-linux-x64.tar.gz -C /opt
配置Java环境变量

安装Hadoop

tar -zxvf hadoop-2.9.2.tar.gz -C /opt

docker run -ti ubuntu:hadoop
